# Data-Analysis-Coursework-R

This repository contains representative R scripts demonstrating core data analysis
methods used across coursework assignments, including:

- Data cleaning and wrangling (tidyverse)
- Exploratory data analysis (EDA)
- Data visualization (ggplot2)
- Summary tables (gtsummary, gt)
- Contingency tables and measures of association (odds ratio)

## Repository Contents
- `scripts/` – Reproducible R scripts demonstrating analytical workflows
- `reports/original_pdfs/` – Original PDF submissions generated in RStudio (reference)

## Note on Reproducibility
Original coursework was submitted as PDF reports. The R scripts in this repository reconstruct representative workflows using publicly available datasets (e.g., MASS::Melanoma).
